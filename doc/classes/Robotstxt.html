allowed?(url, robot_id)

Check if the URL is allowed to be crawled from the current Robot_id. Robots:Allowed? returns true if the robots.txt file does not block the access to the URL. 

 Robotstxt.allowed?('http://www.simonerinzivillo.it/', 'rubytest')
